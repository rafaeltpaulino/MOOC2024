# Python Programming MOOC 2024
Repositório dedicado aos exercícios que eu fiz no curso Python Programming MOOC 2024 da Universidade de Helsinque, Finlândia.
Link do curso: https://programming-24.mooc.fi/

O curso é dividido em 8 partes e é totalmente em Python. O curso abrange muitos conceitos de programação, indo de coisas básicas como estruturas condicionais e laços de repetição, até conceitos mais avançados como estuturas de dados, manipulação de arquivos, recursão, programação orientada à objetos, programação funcional e criação de jogos.
